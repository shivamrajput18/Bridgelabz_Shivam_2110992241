int x=5;  //ordinary variable

int *p;  //pointer variable
    p=&x    pointer variable p stores address of x. 
    
Reference Variable
 int &y=x; 
   y++; // y=6
   cout<<y<<" "<<x;   // 6 6
   
Reference means address
Reference variable is an internal pointer
Declaration of Reference variable is preceded with '&' symbol(but do not reads it as 'addressof')
Reference variable must be initialised during declaration
it can be initialised with already declared variables only.
Reference variables can not be updated.     
Pointer can be updated.
